# AgriTech Mobile (Expo)

Mobile app for Farmer and Agronomist roles, built with Expo + React Native.

## Quick Start

```bash
cd mobile
npm install
npm run start
```

Then:

- Press `i` for iOS Simulator (macOS)
- Press `a` for Android Emulator
- Or scan the QR code with Expo Go

## Notes

- Web admin stays in the existing React app.
- Mobile app includes farmer + agronomist flows only.
